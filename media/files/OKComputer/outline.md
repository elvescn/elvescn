# 118号元素网站项目大纲

## 文件结构

### 核心HTML页面
1. **index.html** - 主页面
2. **periodic-table.html** - 交互式元素周期表
3. **properties.html** - 科学性质详情页

### 资源文件
- **resources/** - 图片和媒体资源文件夹
- **main.js** - 主要JavaScript功能
- **styles.css** - 自定义样式（如需要）

## 页面详细规划

### 1. index.html - 主页面
**目标**：震撼的首页体验，展示Og的独特性

**内容结构**：
- **导航栏**：简洁的顶部导航，包含Logo和主要页面链接
- **Hero区域**：
  - 背景：深色宇宙风格配合粒子效果
  - 中心：大型3D Og原子模型（使用p5.js）
  - 标题：大字体"Oganesson"配合颜色循环动画
  - 副标题："118号元素 - 周期表的终点"
  - 按钮："探索元素"链接到周期表页面

- **基本信息区域**：
  - 左侧：Og的基本数据卡片（原子序数、符号、原子量等）
  - 右侧：发现历史简介和科学家Yuri Oganessian介绍
  - 底部：有趣的事实列表（如"仅制造过5个原子"）

- **独特性质预览**：
  - 3列卡片布局展示Og的独特性质
  - 卡片1："非典型的惰性气体" - 解释其反应性
  - 卡片2："相对论效应" - 电子速度接近光速的影响
  - 卡片3："超重元素" - 原子核的巨大质量

- **发现时间线**：
  - 水平时间轴展示从1999年到2016年的发现历程
  - 交互式节点，点击查看详细信息
  - 使用Anime.js实现流畅的滚动动画

### 2. periodic-table.html - 交互式元素周期表
**目标**：完整的周期表体验，突出Og的位置

**内容结构**：
- **导航栏**：与首页保持一致
- **周期表主体**：
  - 完整的交互式元素周期表
  - Og元素使用特殊样式（发光、脉冲效果）
  - 其他超重元素也使用特殊标记
  - 点击任何元素显示基本信息弹窗

- **筛选功能**：
  - 左侧边栏：元素类型筛选器
  - 按状态（固体、液体、气体、未知）
  - 按发现时间（古代、近代、现代、超重）
  - 按放射性（稳定、放射性、高度放射性）

- **Og详情面板**：
  - 右侧边栏：Og的详细信息
  - 电子构型图解
  - 同位素信息
  - 预测性质数据

- **对比功能**：
  - 选择多个元素进行性质对比
  - 使用ECharts.js创建对比图表
  - 支持熔点、沸点、原子半径等属性对比

### 3. properties.html - 科学性质详情页
**目标**：深入展示Og的科学特性和研究进展

**内容结构**：
- **导航栏**：保持一致性
- **原子结构可视化**：
  - 大型3D原子模型（使用p5.js）
  - 可旋转、缩放
  - 电子层动画效果
  - 相对论效应可视化

- **性质数据表**：
  - 详细的物理和化学性质表格
  - 实验观测值 vs 理论预测值
  - 数据来源和可靠性说明

- **合成方法**：
  - 核反应方程式展示
  - 实验装置图解
  - 合成难度和挑战说明

- **研究应用**：
  - 当前研究进展
  - 未来应用前景
  - 相关研究机构介绍

- **学术资源**：
  - 重要论文链接
  - 研究机构网站
  - 相关科学新闻

## 技术实现要点

### JavaScript功能模块
1. **原子模型渲染** (p5.js) - 3D原子结构可视化
2. **周期表交互** (原生JS) - 元素选择和筛选
3. **数据可视化** (ECharts.js) - 科学图表和对比
4. **动画效果** (Anime.js) - 页面过渡和元素动画
5. **物理模拟** (Matter.js) - 放射性衰变动画
6. **背景特效** (PIXI.js + Shader-park) - 科技感背景

### 响应式设计
- 桌面端：完整功能体验
- 平板端：简化动画，保持核心功能
- 移动端：优化触摸交互，重点内容优先

### 性能优化
- 图片懒加载
- 动画按需加载
- 代码分割和压缩
